Go Language -dudu-





What is that?

Golang (atau biasa disebut dengan Go) adalah bahasa pemrograman baru yang dikembangkan di Google oleh Robert Griesemer, Rob Pike, dan Ken Thompson pada tahun 2007 dan mulai diperkenalkan di publik tahun 2009.
Penciptaan bahasa Golang didasari bahasa C dan C++, oleh karena itu gaya sintaks-nya mirip.

Setelah Belajar Golang

Golang sedikit berbeda dengan bahasa yang telah di pelajari. Walaupun secara algoritma pemakaiannya hampir sama saja. Tapi secara penggunaan sedikit berbeda. Seperti kita bisa import dari sebuah website sebagai src nya. Untuk pemakaiannya pun kita harus punya sebuah folder yang berisikan folder src, package, dan bin. Package berisikan fungsi yang bisa kita panggil, tapi harus kita import terlebih dahulu. Dan sebelum kita bisa ngoding dengan bahasa GO pun kita harus install plugin GOnya, misalnya kalau kita menggunakan visual studio code, kita hanya perlu mencari pluginnya di visual studio code dan menginstallnya, sehingga kita dapat memakainya.

Contoh Penggunaan bahasa GO

program diatas adalah kita memanggil package main, dan mengimport fmt, sehingga kita bisa menggunakan fmt.Println(“hello world”) dimana kita bisa menampilkan hello world di console log.

Conclusion

walaupun memang banyak perbedaan, tetapi penggunaan hampir sama, hanya berbeda bahasanya, bahkan bisa lebih mudah penggunaannya, tergantung bagaimana kita melihatnya. Ini hanya berbeda bahasa, tidak berbeda cara penggunaannya.

Node Js

/f/42126/336x255/b3cd4d7025/nodejs.png)

Node.js adalah bahasa side-server, untuk memakainya kita perlu basic dari javaScript, dengan itu kita bisa menyelam lebih dalam ke bagian front-end dan back-end. Node Js adalah bahasa yang memberikan performa yang luar biasa. Fiturnya adalah memiliki ekosistem yang aktif dengan banyaknya paket pihak ketiga, dimana kita bisa memasukannya ke aplikasi kita. Yang membuatnya populer adalah karena semakin kesini semakin penting. Dan juga pada 2018 juga.



Front-End Frameworks



Framework pada FrontEnd contohnya seperti Angular, React atau Vue.js sangat membantu di bagian client javascript. Framework ini membantu kita untuk bisa membuat user experience yang menarik pada aplikasi web yang dikontrol menggunakan javascript. Ini sangat membantu dalam membuat user experience yang baik entah di mobile atau di pc.

Untuk mempelajarinya lebih dalam kita bisa melihat contohnya dari link-link dibawah :

  2. Angular
  3. Vue.js


Static Page Generators

/f/42126/1039x247/d6879a9479/static-page-generators.png)


Bagaimana kita menyajikan pre-rendered version dari web kita tanpa harus pre-rendering secara dinamik ke suatu server? Jawabannya adalah static page generators. seperti Jekyll, Hugo atau project seperti di Nuxt.js untuk vue.js. Semua itu membantu kita mencapai 1 hal yaitu membuat halaman website yang bisa kita pre-render dalam server lokal kita atau dalam sebuah server.
